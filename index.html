Proposal

The goal is to create a website that has mulitple calculators that can be used to calulate statistics after a game. Instead of searching for the formulas, there will be a site that has calculators at the ready.

Goals of the Site

  1. Have a page and multiple calculators for baseball, hockey, and golf statistics
  2. Make it feel like a sports themed site
  3. Make each sport selection feel themed to that sport

Example Page

I chose this site because it has similar calculators that I plan on recreating. The site has a very old 95 feel to it and I would like to have a site that feels more modern.

Content

Home Page:

The home page will have a description of what the site will be used for. There will be a section for each sport that will have a list of the statistics calculators that I plan on having. The sections will look similar to the sections that are on the example site but they backgrounds will have images of the sport correlating to each sport.

ERA Page:

The ERA page will have a description of what ERA is and how it is measured. There will then be an input box for earned runs and another input box for innings pitched.

Slugging Percentage Page:

The slugging percentage page will have a description of what slugging percentage is and how it is measured. There will then be an input box for home runs, triples, doubles, singles, and outs.

WHIP Page:

The WHIP page will have a description of what WHIP is and how it is measured. There will then be an input box for walks, hits, amd innings pitched.

Batting Average Page:

The batting average page will have a description of what batting average is and how it is measured. There will then be an input for number of hits and number of official at bats.

Audience

The target audience is going to be anyone who wants to watch sports and keeps track of the basic stat such as shots, goals, time played, and etc. They would then take these basic stats and open up this site so they could do quick calculations of stats that require a combination of these basic recorded stats.
